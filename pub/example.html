Getting Started

A library that generates customizable and interactive notifications. The notifications ("notes") can take the form of toast messages, or they can attach themselves to different positions of any element. Notifications can be built easily with the provided components.

Examples

Here are some example usages

Create

Create instance to generate notes:
const noteGen = new Note()

Create a new Note:
// with createNote():
const toast = noteGen.createNote().addText("hello")

// or use templates:
const toast = noteGen.makeToast("hello")

/* the two options above achieve the same result */

Display

Now let's take the toast we created above and display it on screen!

  • You can display the note on the viewport (attach to body): 

    • toast.display("top-center")
    toast.display("top-right")
    toast.display("mid-right")
    toast.display("bot-right")
    toast.display("bot-center")
    toast.display("bot-left")
    toast.display("mid-left")
    toast.display("top-left")
  • Or display the note relative to some element (attached to target element)
    • TARGET

    You can change the duration of display, and duration of zero means the note won't fade out. 

    toast.display("top-center", target, {duration: 0})
    toast.display("mid-right", target)
    toast.display("bot-center", target)
    toast.display("mid-left", target)

Modify

Let's start with an empty note:
const note = noteGen.createNote()

Add some text:
note.addText("Change header color?")

Add some buttons: (refresh the page if you want to try other colors) 
note.addButtons(["orange", "blue", "pink"])
