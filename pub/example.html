Note.js


Library Description:
A library that generates customizable and interactive notifications. The developer can choose where the notification message appears (top, bottom, corner, etc.). The notifications can take the form of toast messages, or they can attach themselves to different positions of any element. The developer can create their notifications from scratch, or use or modify the templates.


Create an instance to generate notes
const noteGen = new Note()
Create a new note with createNote() or using existing templates
const note = noteGen.createNote().addText("hello") or const toast = noteGen.makeToast("hello")

Display the note object you've created (let's take the "toast" above):
1) display the note on the viewport (attach to body):
toast.display("top-center")
toast.display("top-right")
toast.display("mid-right")
toast.display("bot-right")
toast.display("bot-center")
toast.display("bot-left")
toast.display("mid-left")
toast.display("top-left")
TARGET
2) display the note relative to some element (attached to target element)
You can change the duration of display, and duration of zero means the note won't fade out. toast.display("top-center", target, {duration: 0})
toast.display("mid-right", target)
toast.display("bot-center", target)
toast.display("mid-left", target)

Add or Deleting Text/Buttons from your note
Let's start with empty note: const note = noteGen.createNote() Add text: note.addText("What is your favorite color?")
Add buttons: note.addButtons(3, ["red", "blue", "green"])
Of course instead you could also chain the above commands together like this: 
const note = noteGen.createNote()
                    .addText("What is your favorite color?")
                    .addButtons(3, ["red', "blue", "green"])
Remove buttons: note.removeButtons(["red, "blue"])
Remove text: note.removeText(0) //0 is index of the first added text
