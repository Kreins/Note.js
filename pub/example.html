Getting Started

Note.js is a library that generates customizable and interactive notifications. The notifications ("notes") can take the form of toast messages, or they can attach themselves to different positions of any element. Notifications can be built easily with the provided components.

Installation

  • make sure you have jQuery.
  • download note.js and note-styles.css
  • add note.js and note-styles.css to your HTML: 
    <!-- library css-->
<link rel="stylesheet" href="note-styles.css" />

<!-- library js-->
<script type="text/javascript" src="Note.js"></script>

Examples

Here are some example usages

Create

Create instance to generate notes:
const noteGen = new Note()

Create a new Note:
// with createNote():
const toast = noteGen.createNote().addText("hello")

// or use templates:
const toast = noteGen.makeToast("hello")

/* the two options above achieve the same result */

Display

Now let's take the toast we created above and display it on screen!

  • You can display the note on the viewport (attach to body): 

    • toast.display("top-center")
    toast.display("top-right")
    toast.display("mid-right")
    toast.display("bot-right")
    toast.display("bot-center")
    toast.display("bot-left")
    toast.display("mid-left")
    toast.display("top-left")
  • Or display the note relative to some element (attached to target element)
    • TARGET

    You can change the duration of display, and duration of zero means the note won't fade out. 

    toast.display("top-center", target, {duration: 0})
    toast.display("mid-right", target)
    toast.display("bot-center", target)
    toast.display("mid-left", target)

Modify

Let's start with an empty note:
const note = noteGen.createNote()

Add some text:
note.addText("Change header color?")

Add some buttons: (refresh the page if you want to try other colors) 
note.addButtons(["orange", "blue", "pink"])
						
let userResponse = note.waitForResponse();
// waitForResponse() returns a Promise, when the promise is resolved 
// it returns the text of the clicked button.

userResponse.then((res) => {
	$(".header")
	.css("background-color", res)
	.css("transition", "background-color 1s ease");
});

Note

The note will disappear once a button is clicked. Remember you can always attach a note to another note if you want to display multiple polls. (See display() method)


Remove some buttons:
note.removeButtons(["red, "blue"])

Replace the first text:
note..replaceText(0, "There is the only one option!")

Of course you could chain the above commands like so:
const note = noteGen
	.createNote()
	.addText("Change header color?")
	.addButtons(["orange", "blue", "pink"])
	.removeButtons(["orange", "blue"])
	.replaceText(0, "There is the only one option!");

For more modifying options, or information about how a click is handled, please check out the APIs below

APIs

Constructor

First use new keyword to create a note generator: 
const noteGen = new Note()

To create a note instance, use createNote() . The note created will be empty. 
const note = noteGen.createNote()

Methods


note.display(position, [target], [options])

Displays this note on screen.

Parameters
  • position: String

    • Specifies the position of the note relative to the target. The available positions are: "top-left", "top-center", "top-right", "mid-left", "mid-right", "bot-left", "bot-center", "bot-right"

  • target (optional): Object/String.

    • default: "body"

    Specifies the target element the note attaches to. If the target is body, then target should take the string "body", or it could be omitted if the [options] parameter is empty. Otherwise, target takes an Element Object.

  • options: Object.

    • default: {duration: 3000}

    Object that contains optional note settings, currently only support changing the note's duration of display. Setting duration to 0 means the note will not fadeout unless an action(click) occurs.

Returns

No return value



note.addText(text)

Add a text node to the bottom of the note

Parameters
  • text: String
    • The text to be added.
Returns
  • note: Note
    • Updated note instance


note.replaceText(index, newText)

Replace a text node at this index

Parameters
  • index: Number
    • Index of the text node to be replaced. The index refers to the order in which the text nodes were added. The top text note would have index of 0.
  • newText: String
    • New text to replace with.
Returns
  • note: Note
    • Updated note instance


note.removeText(index)

Remove the text node at this index

Parameters
  • index: Number
    • Index of the text node to be removed. The index refers to the order in which the text nodes were added. The top text note would have index of 0.
Returns
  • note: Note
    • Updated note instance


note.addButtons(buttonTexts)

Add buttons to the note.

Parameters
  • buttonTexts: array of String
    • An array of the buttons' text.
Returns
  • note: Note
    • Updated note instance


note.replaceButton(index, newText)

Replace the button at this index.

Parameters
  • index: Number
    • Index of button to be replaced.
  • newText: String
    • The new button's text.
Returns
  • note: Note
    • Updated note instance


note.removeButtons(buttonTexts)

Remove buttons from this note

Parameters
  • buttonTexts: array of String
    • For each String in this array, find and remove the button that contains this String.
Returns
  • note: Note
    • Updated note instance


note.waitForResponse()

Waits for a button in this note to be clicked.

Returns
    Returns a Promise. When a button is clicked, the Promise will resolve and the button's text will be returned.
Example
// create a note with two buttons: "yes", "no"
const note = noteGen.createNote().addButtons(["yes", "no"])

let result = note.waitForResponse()
result.then((res) => console.log(res))
// logs "yes" or "no" depending on the button clicked
